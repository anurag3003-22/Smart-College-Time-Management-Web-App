# Smart College Time Management Web Application

This project is a **Smart College Time Management Web Application** designed to help
students manage their study time, assignments, exams, and daily schedules efficiently.
The system generates **daily and weekly study plans** based on user inputs such as
subjects, deadlines, and available time, helping students stay organized and productive.

---

## Problem Statement
College students often struggle with managing multiple subjects, assignments,
and exam preparation due to poor time planning and lack of proper guidance.
This leads to stress, missed deadlines, and inefficient study routines.

---

## Solution
The application provides an intelligent time management system that generates
personalized daily and weekly study schedules based on student inputs.
It helps students prioritize tasks and utilize their available time effectively.

---

## Features
* Daily and weekly study plan generation
* Assignment and exam deadline tracking
* Priority-based task management
* Simple and user-friendly web interface

---

## Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Java
* **Tools:** VS Code, Git, GitHub
* **Logic:** AI-based smart planning approach
